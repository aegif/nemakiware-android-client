Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => app\src\main\AndroidManifest.xml
* libs\commons-io-1.4-javadoc.jar => app\libs\commons-io-1.4-javadoc.jar
* libs\commons-io-1.4-sources.jar => app\libs\commons-io-1.4-sources.jar
* libs\commons-io-1.4.jar => app\libs\commons-io-1.4.jar
* libs\dom4j.jar => app\libs\dom4j.jar
* res\ => app\src\main\res\
* src\ => app\src\main\java\
